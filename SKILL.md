# Daily News Skill

每日自动抓取新闻和行业动态，结合半导体芯片行业资讯，生成简洁日报并推送。

## 四大板块

1. **🔥 芯海直击** — 芯海科技股价、公告、产品进展。同时覆盖：
   - 下游客户芯片相关动态：华为、大疆、联想、小米、荣耀
   - 上游供应商动态：中芯国际、华虹半导体
   - 只保留芯片/供应链相关的新闻，不收录客户公司的非芯片动态
2. **📋 政策雷达** — 以下机构发布的科技/产业政策、**人才政策**，只保留与半导体芯片或**芯片产业人才**相关的内容：
   - 国家级：工信部、发改委、科技部
   - 省级：广东省工信厅、四川省经信厅/科技厅、安徽省经信厅/科技厅
   - 市级：深圳市科创局/工信局/发改委、成都市科创局、合肥市科技局
   - 区级/园区：深圳市南山区、成都高新区管委会、合肥高新区管委会
3. **🔬 行业前沿** — 从Kevin的VP+主权架构师身份出发，关注：
   - **竞争格局** — 直接竞品动态：ST/NXP/兆易/中颖等MCU厂商、车规芯片竞争者
   - **下游市场** — 消费电子/工业控制/汽车电子/AIoT 等应用领域的需求变化与趋势
   - **技术路线** — 与芯海业务直接相关的技术方向（高精度ADC、低功耗、无线连接等）
   - **产业链** — 晶圆代工/封测/产能变化，上游材料/设备动态
4. **💰 资本温度** — 板块行情、IPO、监管动态、并购事件、投融资

## 推送规则

- **简要摘要+来源链接** — 每条新闻写简要摘要，末尾附原始新闻链接
- **使用真实数据** — 确保时效性和来源可追溯
- **当天新闻优先** — 如果当天没有相关新闻，不推送该板块
- **推送渠道** — 推送到飞书对话（每天 07:00 Cron 自动执行）
- **存档到飞书云盘** — 同时创建飞书 Docx 文档保存到「每日日报」文件夹（参见 references/feishu-doc-archive.md）

## 输出格式

每条新闻写简要摘要，末尾附原始新闻链接，链接用纯文本（不用Markdown超链接 `[文字](url)`）。使用真实数据，确保时效性。

### Feishu 推送格式规范

**⚠️ Feishu 显示限制**：飞书消息体中过多空行和冗余格式会导致显示凌乱。适用以下规则：

1. **紧凑布局** — 板块之间只保留1个空行，标题与正文之间不空行
2. **标题加粗** — 四大板块标题使用 `**` 加粗，后接换行，然后直接接列表
3. **无多余装饰** — 不使用表格式布局、额外引用块、分割线
4. **无前缀后缀** — 不要在标题前加额外符号，不要加 "以上就是今日日报" 之类结尾语

标准结构如下（紧凑版）：

```
📰 **今日新闻日报 · YYYY年M月D日**
**🔥 芯海直击**
- 简要摘要 原始链接
**📋 政策雷达**
- 简要摘要 原始链接
**🔬 行业前沿**
- 简要摘要 原始链接
**💰 资本温度**
- 简要摘要 原始链接
```

### 对比示例

📰 **今日新闻日报 · 2026年6月3日**

**🔥 芯海直击**
- 芯海动态...

## ⚠️ 已知陷阱（关键）

### 陷阱1：新闻日期核查 — 网站"今天发布"不等于事件发生在今天
**问题（2026-06-07实错）：** 爱集微/财联社显示"今天发布"的芯海科技新闻（参展COMPUTEX、ASPICE认证等），实际事件发生在6月2日即更早，是旧闻被网站重新推送。

**铁律：**
- 每条新闻必须核实原始事件发生日期
- 新闻报道如果涉及的是更早日期的事件，不算当天新闻
- 只收录原始事件发生在当天的新闻
- 芯海科技股价数据（收盘价）可作为当日新闻

**识别信号：** 新闻提到"近日""日前""上周"等非具体日期词语 = 可能不是当天事件，需进一步核实源新闻的发布时间而非聚合页显示时间。

## 已知陷阱

### 爱集微搜索URL参数不可用（2026-06-05）
**问题**：直接访问 `https://www.laoyaoba.com/search?q=芯海科技` 不会返回搜索结果，而是显示首页推荐内容。
**正确操作**：必须点击页面右上角「搜索」图标 → 在弹出的搜索框中输入关键词 → 点击「站内搜索」按钮。
**验证**：搜索成功后的URL签名包含会话标识，不可手动拼接。

### 百度/Google CAPTCHA阻塞（2026-06-05）
**问题**：自动化浏览器访问百度或Google搜索时，频繁弹出CAPTCHA验证码，导致无法获取搜索结果。
**推荐替代**：优先使用财联社搜索页（`searchPage?keyword=半导体&type=telegraph`）和爱集微站内搜索替代通用搜索引擎。

### 财联社搜索页结果未自动显示（2026-06-06）
**问题**：直接访问 `searchPage?keyword=半导体&type=telegraph` 后，搜索框虽已预填关键词，但实际电报内容未显示在页面上。
**原因**：该搜索页的搜索结果区域默认未渲染，需手动触发过滤器。
**正确操作**：在搜索结果页内找到「电报」Tab（显示类似「电报(26123)」的计数），点击该Tab后内容才会呈现。注意此处「电报」Tab是**搜索页结果筛选器**，并非顶部导航栏的「电报」链接。
**验证**：点击后页面应展示带时间戳的电报条目列表。

### Feishu DM 交付丢失（2026-06-03）
**问题**：Cron job 配置 `deliver: origin` 在飞书 DM 会话中会丢失上下文目标，导致定时任务执行后只保存本地文件，未推送到飞书。

**表现**：Cron job 状态显示 `last_status: ok`，输出文件存在于 `~/.hermes/cron/output/<job_id>/`，但用户未收到飞书消息。

**修复**：将 cron job 的 `deliver` 参数从 `origin` 改为明确的飞书对话 ID：
```yaml
deliver: feishu:oc_c0d7ced8301a8c690beccf11ba7ee97d
```

**验证**：
```bash
# 检查 cron job 配置
hermes cron list | grep "日报自动发布"

# 手动测试发送
hermes send feishu:oc_c0d7ced8301a8c690beccf11ba7ee97d "测试消息"
```

可用对话ID可通过 `send_message(action='list')` 查询。
