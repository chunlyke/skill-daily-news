# Daily News Skill

每日自动抓取新闻和行业动态，结合公司（MCU/芯片）相关资讯，生成简洁日报并推送。

## 功能

- **每日头条** — 今日最重要的半导体/MCU行业新闻
- **行业动态** — 芯片行业最新动态与技术趋势
- **公司相关** — 芯海科技及主要竞品/合作伙伴相关新闻
- **资本市场** — 半导体板块行情、IPO、融资事件

## 输出格式

每条新闻一句话概括核心信息，不加点评和多余分析。使用真实数据，确保时效性。

## 调用方式

直接作为 Cron 任务运行，每天 07:00 自动执行并推送到飞书。

```bash
# 手动运行
hermes cron run <job_id>
```
