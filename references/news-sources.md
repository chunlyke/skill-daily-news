# 每日新闻来源与搜索操作

## 来源清单

### 1. 财联社电报（实时快讯）
**URL:** `https://www.cls.cn/telegraph`
**操作步骤：**
1. `browser_navigate(url="https://www.cls.cn/telegraph")`
2. **首选方法：使用搜索框**
   - 从 snapshot 找到搜索框（textbox "请输入要搜索的信息"）
   - `browser_type(ref=..., text="半导体")` 然后 `browser_press(key="Enter")`
   - 搜索结果按时间排序，可切换到「电报」Tab 查看实时电报
3. **备选方法：点击分类标签**（若页面有「半导体芯片」标签）
   - 部分页面版本可能不显示分类标签，搜索框更可靠
4. 抓取页面显示的当日新闻条目

**适用板块：** 资本温度、芯海直击
**特点：** 24小时实时更新，适合抓取当天最新消息和A股公告
**搜索技巧：** 搜索结果页面会显示「电报(数量)」「资讯(数量)」等分类Tab，点击「电报」可聚焦实时快讯

### 2. 爱集微（半导体行业垂直媒体）
**URL:** `https://www.laoyaoba.com/`
**操作步骤：**
1. `browser_navigate(url="https://www.laoyaoba.com/")`
2. **首选：点击导航栏「最新」Tab** → 页面加载后，注意出现的**分类筛选按钮栏**（位于文章列表上方，包含：全部、老杳吧、学研、IC、本土IC、科技）
   - 点击「IC」按钮 → 只显示芯片/半导体类文章
   - 点击「本土IC」按钮 → 显示国产芯片相关文章
   - 点击「科技」按钮 → 显示科技行业综合文章
   - **效率提示**：先点「IC」或「本土IC」再浏览，大幅减少无关内容噪音
3. **备选：点击导航栏「概念股」Tab** → 展示资本市场/概念股相关文章，适合「资本市场」板块素材收集
4. **搜索特定公司：**
   - 点击页面右上角「搜索」图标
   - 在弹出的搜索框中输入公司名如「芯海科技」
   - 点击「站内搜索」按钮（注意：URL参数方式不可用，搜索成功后的URL含会话标识）
   - 搜索结果页面有「文章」「视频」「专题」筛选Tab，默认显示文章
   - 注意：搜索结果中可能混入旧日期的公司官网公告类文章，优先选取近一周内带明确时间戳的内容
5. 滚动页面获取更多文章列表，底部有「查看更多」按钮可加载更多
6. 尤其关注标注「本土IC」「概念股」「IC」标签的文章

**适用板块：** 行业前沿、芯海直击
**特点：** 深度行业分析，覆盖芯海科技、纳芯微、泰凌微等国产芯片公司
**搜索技巧：** 搜索芯海科技时，爱集微文章通常有明确发布时间标签，优先选取近一周内的文章

### 3. 补充搜索（百度）⚠️ 已知陷阱
当上述来源缺少特定公司新闻时，尝试百度搜索（注意：**频繁触发CAPTCHA验证码**，需做好备选方案）：
- `芯海科技 2026年6月`
- `MCU 芯片 最新动态`
- `半导体 资本市场`

**已知陷阱：** 百度和Google均会在无痕/自动化环境下弹出CAPTCHA验证。**推荐替代方案：** 使用财联社搜索页（方案4）和爱集微站内搜索替代通用搜索引擎。

### 4. 财联社搜索页（半导体专属，推荐方案）
**URL:** `https://www.cls.cn/searchPage?keyword=半导体&type=telegraph`
**操作步骤：**
1. `browser_navigate(url="https://www.cls.cn/searchPage?keyword=半导体&type=telegraph")`
2. **关键步骤：页面加载后，先不要急着抓取**。搜索页默认不渲染内容列表，需要在页面内找到「电报」Tab（显示类似「电报(26123)」的计数链接），点击该Tab
3. 点击后页面展示该关键词相关的实时电报，按时间倒序排列
4. 也可点击「资讯」Tab 查看深度文章
5. 如需搜索其他关键词，修改URL参数：`keyword=芯海科技`

**⚠️ 操作要点**：此处的「电报」Tab是搜索结果页内的**结果筛选器**（位于搜索框正下方），并非顶部导航栏的「电报」链接。必须区分两者。

**优势：** 比在首页找分类标签更可靠，CAPTCHA风险低，搜索结果按时间精确排序
**适用板块：** 行业前沿、资本温度

### 7. 爱集微「概念股」板块
**URL:** `https://www.laoyaoba.com/`
**操作步骤：**
1. **快速获取：直接查看首页主内容区** — 首页主内容区已按时间倒序列出最新文章，每条附有摘要、标签（如「概念股」标签）和发布时间，无需额外点击即可获取近期行业动态

4. **客户机型线索** — 芯海科技相关文章中，首页摘要中常直接提及「联想、小米、荣耀」等客户终端机型，对「公司相关」板块有价值

**适用板块：** 行业前沿、资本市场、公司相关
**效率提示：** 先看首页主内容区获取最近文章摘要，发现高价值文章后再点击进入详情页获取完整信息，避免一开始就做站内搜索

**适用板块：** 公司相关
**优势：** 相比从首页搜索（方案6），直接访问搜索页URL更快，结果直接展示

## 输出规范

### 格式模板（v2.0.0）
```
📰 **今日新闻日报 · YYYY年M月D日**
**🔥 芯海直击**
- 简要摘要 原始链接
**📋 政策雷达**
- 简要摘要 原始链接
**🔬 行业前沿**
- 简要摘要 原始链接
**💰 资本温度**
- 简要摘要 原始链接
```

### 输出规则（v2.0.0）
- 每条写简要摘要，末尾附原始新闻链接
- 只陈述事实，不加引导语
- 当天新闻优先，当天没新闻不推送该板块
- 飞书推送紧凑版：板块间1个空行，标题加粗后直接接列表，不加结尾语和分割线
