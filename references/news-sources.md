# 新闻来源与操作细节

## 第一部分：核心源（每日必扫）

### 1. 财联社电报 — 半导体分类
- URL: `https://www.cls.cn/telegraph`
- 操作：页面顶部有分类标签栏（全部/加红/公司/看盘/港美股/基金/提醒），注意没有显式的"半导体芯片"分类
- 搜索替代方案：使用搜索页 `https://www.cls.cn/searchPage?keyword=半导体&type=telegraph`
- **关键步骤**：进入搜索页后，必须点击「电报」Tab（搜索结果筛选器）才能显示带时间戳的电报条目。URL参数虽已预填keyword但内容默认不渲染

### 2. 爱集微 — 最新文章 + 分类筛选
- URL: `https://www.laoyaoba.com/`
- 首页主内容区直接展示最新概念股文章（含摘要）
- 「最新」Tab → 分类筛选按钮栏包含「全部」「老杳吧」「学研」「IC」「本土IC」「科技」等按钮
- 「概念股」Tab → 专门展示资本市场/概念股相关文章

#### 爱集微"概念股"Tab的日期验证优势
- 文章列表显示**显式时间戳**格式如 `06-06 22:47`（而非"9小时前"等相对时间）
- 这使其成为验证新闻原始日期的最佳来源，比"最新"Tab（显示相对时间）更可靠
- 周六/周日该Tab持续产出行业文章，可优先使用

#### 爱集微站内搜索（URL参数不可用）
- 正确操作：点击右上角「搜索」图标 → 在弹出的搜索框中输入关键词 → 点击「站内搜索」按钮
- 搜索成功后的URL包含会话签名，不可手动拼接

### 3. 财联社搜索（推荐优先使用）
- URL: `https://www.cls.cn/searchPage?keyword=半导体&type=telegraph`
- 直接搜索半导体相关实时电报
- 支持多关键词组合如 `keyword=芯海科技`、`keyword=MCU+芯片`
- **必须点击「电报」Tab** 才会显示结果（参考陷阱说明）

### 4. 新浪财经（芯海科技股价）
- URL: `https://finance.sina.com.cn/realstock/company/sh688595/nc.shtml`
- 获取实时股价、市值、成交量等数据
- 页面内容庞大，snapshot可能被截断，需滚屏或分区域读取
- 周六/周日显示周五收盘数据

1. 确认 WeChat 在 MacMini 运行中：`ps aux | grep WeChat | grep -v grep`
2. 截屏：`screencapture -x /tmp/wechat.png`
3. 分析：通过 vision_analyze 读取微信左侧聊天列表中的公众号名称
4. 滚动查看更多：用 `osascript` 发送 Page Down 键模拟滚屏
5. 记录完整列表，更新到本文件

**局限性**：WeChat 使用 WeChatAppEx 框架，该框架不暴露 AX UI 元素给 System Events，AppleScript 无法直接读取列表内容，只能通过截屏+视觉识别。

## 第五部分：已知陷阱汇总

### 陷阱A：爱集微搜索URL参数不可用
直接访问 `https://www.laoyaoba.com/search?q=芯海科技` 不会返回搜索结果。
→ 必须通过页面内搜索框操作。

### 陷阱B：百度/Google CAPTCHA阻塞
自动化浏览器访问百度/Google搜索时频繁弹出验证码。
→ 优先使用财联社搜索页和爱集微站内搜索。

### 陷阱C：财联社搜索页结果未自动显示
→ 必须点击「电报」Tab筛选器。

### 陷阱D：新闻日期核查 — 网站"今天发布"不等于事件发生在今天
→ 每条新闻必须核实原始事件发生日期。爱集微概念股Tab的显式时间戳（如"06-06 18:27"）仅表示文章发布/更新时间，事件本身的日期需阅读正文确认。新闻报道中若出现"近日""日前""上周"等非具体日期词语则为警示信号。

### 陷阱E：搜狗微信反爬
搜狗微信搜索（`weixin.sogou.com`）频繁请求会触发验证码。
→ 仅作为公众号文章的补充检索手段，不要作为每日主来源。

### 陷阱F：TrendForce付费墙
TrendForce的大部分深度研报需要付费订阅。
→ 只取首页摘要/免费内容，付费报告不强行访问。

### 陷阱G：Digitimes订阅限制
部分Digitimes深度文章需要订阅会员才能查看全文。
→ 取标题+摘要即可，不强行绕过付费。
